Privacy policy

WindowSnap collects nothing.

Effective date: July 13, 2026

WindowSnap collects no data. It has no telemetry, no analytics, no accounts, and it never connects to the internet.

WindowSnap is a lightweight Windows utility that lets you move and resize application windows into screen slots using keyboard shortcuts. It runs entirely on your computer.

Data collection

WindowSnap does not collect, store, transmit, sell, or share any personal data or usage data. Specifically, it has:

  • No telemetry or analytics
  • No crash reporting
  • No advertising or tracking
  • No user accounts or sign-in
  • No network access of any kind

Local data

The only data WindowSnap writes is its own settings file, stored on your computer at ~/.windowsnap/config.toml. It contains your hotkey and layout preferences and nothing else. It never leaves your machine, and you can delete it at any time.

Window information

To do its job, WindowSnap uses standard Windows APIs to list open windows and their titles so it can match and move them. This information is processed in memory only — it is never logged, saved, or transmitted anywhere.

Third-party services

WindowSnap uses no third-party services, SDKs, or libraries that collect data.

Children's privacy

WindowSnap does not collect data from anyone, including children.

Changes to this policy

If this policy ever changes — for example, if a future version adds an online feature — the updated policy will be posted at this page with a new effective date.

Contact

Questions about this policy? Open an issue on the WindowSnap GitHub repository.
